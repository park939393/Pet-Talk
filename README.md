# 프로젝트 명 : Pet Talk

## 인원 : 5명
## 기간 : 2023.03.15 ~ 2023.03.21
## 기능 
### 유저 - 회원 가입, 로그인, 나의 정보, 반려 동물 정보 입력, 친구 추가 기능
### 어드민 - 회원 관리 기능
### 채팅 - 다중 접속 채팅, 귓속말 기능
### 그림판 - 그림 그리기 기능
### 동물 병원 정보 찾기
## 개발 언어 : Java 11
## 개발 환경 : Eclipse
## 데이터베이스 : MySQL
## 간단 소개 : 반려 동물의 사회성을 기르고 견주들의 친목을 위한 채팅 프로그램
